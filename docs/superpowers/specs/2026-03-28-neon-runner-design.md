# Neon Runner - 游戏设计规范

## 1. 项目概述

**项目名称：** Neon Runner
**项目类型：** 休闲游戏 - 动作类平台跳跃战斗
**技术栈：** Python + Pygame
**用途：** 比赛参赛作品

## 2. 视觉设计

### 配色方案

| 元素 | 颜色 |
|------|------|
| 背景 | #0a0a1a (深蓝黑) |
| 网格线 | #1a1a3a (暗蓝) |
| 玩家核心 | #ffffff (白色) |
| 玩家轮廓 | #00ffff (青色) |
| 敌人核心 | #ff0000 (红色) |
| 敌人轮廓 | #ff00ff (品红) |
| 平台基底 | #1a1a2e (深紫蓝) |
| 平台边缘 | #0088ff (霓虹蓝) |

| 子弹 | #00ffff (青色) |

### 视觉效果

- 背景网格消失点效果
- 角色落地粒子爆炸
- 跳跃拖尾残影
- 攻击屏幕震动
- 敌人死亡碎片四散
- 背景网格缓慢滚动
- 角色/敌人发光轮廓

## 3. 游戏玩法

### 控制方案

| 功能 | 按键 |
|------|------|
| 向左移动 | ← 方向键 或 A 键 |
| 向右移动 | → 方向键 或 D 键 |
| 跳跃 | 空格键 (二段跳) |
| 攻击 | 鼠标左键 (追踪光束) |

### 游戏机制

- **生命系统：** 3颗心，被攻击后短暂无敌
- **跳跃：** 支持二段跳
- **攻击：** 发射追踪光束
- **平台：** 静态平台 + 碰撞检测

### 敌人设计

- 简单AI：追踪玩家
- 接触伤害：碰到敌人扣血
- 死亡效果：碎片四散

## 4. 项目结构

```
game/
├── main.py          # 游戏入口
├── player.py        # 玩家角色
├── enemy.py         # 敌人类
├── platform.py      # 平台类
├── projectile.py    # 子弹类
├── particles.py     # 粒子系统
├── constants.py     # 颜色/尺寸常量
└── levels/
    └── level1.py    # 第一关卡设计
```

## 5. 开发阶段

1. **基础框架** — Pygame初始化、游戏循环、窗口设置
2. **角色系统** — 玩家移动、跳跃、重力物理
3. **平台系统** — 静态平台、碰撞检测
4. **敌人系统** — 敌人生成、AI移动、攻击
5. **粒子特效** — 视觉增强
6. **关卡打磨** — 调整难度曲线、添加UI
