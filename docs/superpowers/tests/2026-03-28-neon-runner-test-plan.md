## 7. 已知问题

1. **平台颜色偏差**：平台基底色 (26, 26, 46) 与设计规范 (26, 26, 46) 一致，但设计规范写的是 #1a1a2e，实际为 (26, 26, 46)
2. **子弹屏幕外处理缺失**：子弹飞出屏幕后未被移除，长期运行可能积累
3. **玩家死亡无处理**：生命值归零后游戏继续运行
4. **缺少关卡文件**：规范中提到 levels/level1.py 但未实现
5. **缺少音效**：规范中未提及但行业标准应有
